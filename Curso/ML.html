MACHINE LEARNING

Conceito, Algoritmos, Técnicas e Aplicações



Resumo:


Machine learning e o aprendizado de máquina, que tem como objetivo treinar uma máquina para realizar alguma tarefa especifica, por exemplo, reconhecer uma pessoa em uma foto, reconhecer transações fraudulentas, fazer diagnósticos médicos, etc.

Para treinar, uma máquina é necessário um grande volume de dados, assim é possível treina- lá com base em padrões e controlar suas respostas.

Existem diversos algoritmos de Machine Learning como Regressão Linear e Logística, Árvores de Decisão e Redes Neurais Artificiais.

Técnicas mais famosa é Aprendizado Supervisionado: Onde os algoritmos são treinados em um conjunto de dados rotulado, onde cada exemplo de entrada está associado a uma saída desejada. O objetivo é aprender a mapear as entradas para as saídas.

Aplicação mais famosa é Reconhecimento de Padrões: Reconhecimento de voz, reconhecimento de objetos em imagens, detecção de fraudes em transações financeiras.
